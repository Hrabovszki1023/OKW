MemorizeCaption( FN, MemKey )

MemorizeCaption( FN, MemKey ) Schlüsselwort Speichert die Caption des FN -GUI-Objektes unter dem gegebenen MemKey.

Parameter

Parameter Description
FN Funktionaler Name des GUI-Objektes
MemKey Bezeichner des Schlüsseles unter dem die Caption gespeichert wird.

Beispiele

Normale Verwendung

EN.SelectWindow( "Login" );
EN.MemorizeCaption( "OK", "MemOK" );

Schlüsselwortausführung ignorieren

EN.SelectWindow( "Login" );
EN.VerifyCaption( "OK", "" );
EN.VerifyCaption( "OK", "${IGNORE}" );
