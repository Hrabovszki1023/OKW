Glossary

A

AN

Keyword parameter: Application Name - Functional name of the application

C

Caption

Die Beschriftung Caption ist der sichtbare und prinzipiel nicht veränderbare Text eines GUI-Objektes. Z.B. der sichtbare Text einer Schaltfläche oder eines Links.

E

ExpVal

Schlüsselwort Parameter: In Verify-Schlüsselwörtern ist Expected Value der erwartet Sollwert der Prüfung.

F

Frame

  • is a description of the GUI
  • which maps the FN to a GUI-Adapter
  • holds the locator for the Object recognition
  • definition of object specific GUI-adapter methods

FN

Keyword parameter: FunctionalName of the GUI-object, e.g. name of main-window or child-window.

K

Keyword

A keyword is a project and technology-independent description of a user activity.

G

GUI-Adapter

Noch zu erledigen:
Description for GUI-Adapter

M

MemKey

Symbolicname for a user defined name created in Memorize-keywords to save GUI-specific values. ${MemKey} gets the value of saved MemKey.

S

SQN

Keyword parameter, SEQuenceName.

SEQ_ID

Keyword parameter, SEQence ID.

Sequence

A sequence is a reusable sequential arrangement of Keyword s.

T

TCN

OKW Variable: TestCaseName. See: ${TCN}
