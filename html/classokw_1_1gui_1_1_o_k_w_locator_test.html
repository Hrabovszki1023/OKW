◆ TC_DynamicLocators()

void okw.gui.OKWLocatorTest.TC_DynamicLocators ( )

Prüft, ob dynamische Locatoren möglich sind.

Dynamischer Locator bedeutet, dass der Locator zur Laufzeig geändert bzw. angepasst werden kann. So kann z.B. ein angezeiter Wert für die Objekterkennung harangezogen werden.
