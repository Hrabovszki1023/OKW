Testfall-Schlüsselwörter

BeginTest( TCN )
Steht als erstes in einem Testfall.
Der aktuelle Name des Testfalls wird in ${TCN} abgelegt
EndTest( )
Markiert das Ende eines Testfalls

Constants and Variables

Constants and variables can be used in all keywords with Val and ExpVal.
${TCN}
Contains the current test case name, if set with BeginTest( TCN ).
"" and ${IGNORE}
A OKW-keyword is skipped when "" or ${IGNORE} is entered as a value.
This constant is important when re-usable sequences. With this, the activities of the OKW-keywords can be turned off in data-driven test.
